# Centro de Operaciones ERP-EDUCATIVA

## Propósito

Este documento centraliza la información operativa, de seguridad y de gestión de riesgos del ERP-EDUCATIVA. Sirve como punto de referencia para el equipo de soporte, administradores de sistemas y auditores.

## Contenido Principal

- **Seguridad Operativa** – políticas y procesos de seguridad.
- **Matriz de Riesgos** – identificación, evaluación y planes de mitigación.
- **Inventario de Activos** – listado de componentes críticos, responsables y accesos.

## Uso

- Actualizar trimestralmente o cuando se introduzcan cambios significativos en la infraestructura.
- Compartir con equipos de soporte y auditoría para garantizar la continuidad del negocio.
